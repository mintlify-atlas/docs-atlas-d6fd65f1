## Getting Help

### Where to Get Support

Depending on your needs, choose the best channel:

#### Discord (Recommended for Quick Help)

- Real-time chat with community members
- Faster responses for general questions
- Great for troubleshooting and tips
- Active community ready to help

[Join Discord →](https://discord.gg/kRZBPpkgwq)

#### GitHub Issues (For Bugs and Feature Requests)

- Detailed bug reports
- Feature requests and enhancements
- Technical discussions
- Track progress on fixes

[Create an Issue →](https://github.com/antinomyhq/forge/issues/new/choose)

#### Documentation (Self-Service)

- Comprehensive guides and tutorials
- API reference
- Configuration examples
- Troubleshooting guides

[Browse Docs →](/introduction)

### Support Channels

| Channel | Best For | Response Time |
|---------|----------|---------------|
| Discord | Quick questions, general help, troubleshooting | Minutes to hours |
| GitHub Issues | Bug reports, feature requests, technical issues | Hours to days |
| Documentation | Self-service learning, reference | Immediate |
| GitHub Discussions | In-depth discussions, ideas, Q&A | Days |

## Community Guidelines

### Be Respectful

- Treat everyone with respect and kindness
- Welcome newcomers and help them learn
- Be patient with questions
- Assume good intentions

### Be Constructive

- Provide helpful feedback
- Explain your reasoning
- Suggest alternatives
- Focus on solutions

### Be Inclusive

- Use inclusive language
- Respect diverse perspectives
- Create a welcoming environment
- Help others feel included

### Be Collaborative

- Share knowledge and expertise
- Work together on solutions
- Accept constructive criticism
- Learn from each other
